# Linear Agent ships to GA

Linear Agent moved from beta to general availability today.

The 25,000 companies on Linear can now point agents at their Cycles. The agents read the same queue as the team, follow the same labels, and ship to the same roadmap. One source of truth for engineers and agents.

What changed in GA:

- Agents work inside Cycles, not adjacent to them. No new tool to learn. The same project filter you already use.
- Roadmap visibility for agent work. PMs see what's queued, what's in flight, and what blocked.
- Audit trail on every agent change. The git blame works the same way it always has.

The team that wrote this built it inside Linear. We had four engineers, six weeks, one Cycle of dogfooding before GA. The day we cut over, our own backlog moved 30% faster.

If you want to try it, open Cycles → Agents in your workspace. Existing teams keep working as is.
